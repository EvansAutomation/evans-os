Connect Instagram Graph API

Follow these steps to auto-sync your Instagram posts. Takes ~15 minutes.

1
Create a Facebook Developer App
Go to developers.facebook.com → My Apps → Create App → Select "Business" type.
2
Add Instagram Graph API Product
In your app dashboard → Add Product → Instagram Graph API. Connect your Instagram Business account.
3
Generate a Long-Lived Access Token
Use the Graph API Explorer to get an access token with instagram_basic, instagram_manage_insights permissions. Exchange for a 60-day long-lived token.
4
Save your token in Supabase
Add a row to the settings table: key = 'ig_access_token'. The Sync button will appear on this page automatically.
